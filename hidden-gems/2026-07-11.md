## ayoubnabil/aiondb

- <https://github.com/ayoubnabil/aiondb>
- 언어: Rust
- 스타: 48
- 생성일: 2026-05-13

PostgreSQL 와이어 프로토콜과 호환되는 SQL 데이터베이스를 Rust로 처음부터 구현한 프로젝트입니다. 자체 스토리지 엔진 위에 쿼리 옵티마이저와 익스큐터를 두고, HNSW·IVF-flat 같은 벡터 인덱스와 그래프(노드/엣지) 접근 경로까지 관계형 테이블 위에 얹은 구조가 특징입니다. v0.3 알파 단계라 "프로덕션 데이터에는 아직 쓰지 말라"고 명시하면서도, 시작 가이드·SQL/벡터 레퍼런스·아키텍처 문서·재현 가능한 벤치마크·Docker 툴링까지 갖춰놓아, DB 엔진을 설계할 때 무엇을 문서화해야 하는지 참고하기 좋습니다.

## LPC4/Full-Stack

- <https://github.com/LPC4/Full-Stack>
- 언어: Rust
- 스타: 87
- 생성일: 2026-04-13

렉서·파서·타입체커·SSA IR 생성기로 이어지는 컴파일러 프론트엔드부터, 레지스터 할당과 페흘홀 최적화를 포함한 RISC-V 코드 생성 백엔드, 어셈블러·ELF-64 링커, 캐시·MMU·특권 모드를 갖춘 5단계 파이프라인 RISC-V VM까지 전 과정을 혼자 Rust로 구현했습니다. 더 나아가 이 VM 위에서 셸과 파일시스템을 갖춘 자체 OS가 부팅되고, 그 안에서 자체 컴파일러·어셈블러·링커로 self-hosting까지 됩니다. 유닛 테스트, 스냅샷 테스트, VM 실행 검증, 커널 통합 테스트까지 갖춘 걸 보면 "장난감 컴파일러"를 넘어서는 엔지니어링 밀도를 보여줍니다.

## chicoxyzzy/cynic

- <https://github.com/chicoxyzzy/cynic>
- 언어: Zig
- 스타: 19
- 생성일: 2026-05-05

레거시 기능과 sloppy mode를 의도적으로 배제한 strict-mode 전용 JavaScript/WebAssembly 엔진을 Zig로 처음부터 짠 프로젝트입니다. NaN-boxing 기반 바이트코드 인터프리터(Lantern), mark-sweep GC(Metla), 스펙 준수 정규식 엔진(Perlex), test262 커맨드를 58,779/58,779 통과하는 WASM 인터프리터(Sarcasm), 기본 활성화된 베이스라인 JIT(Bistromath)까지 컴포넌트별로 이름까지 붙여가며 구성했습니다. pre-alpha 단계지만 병렬 test262 하니스, 상세한 로드맵 문서, 브라우저용 플레이그라운드까지 갖춰 "취미로 만든 JS 엔진" 치고는 검증 체계가 상당히 진지합니다.
