# 미분과 기울기 (Derivatives and Gradients)

## 한 줄 요약

미분은 "그 점에서 함수가 얼마나 가파른가"(순간 변화율)이고, 다변수로 확장한 기울기(gradient)는 "가장 가파르게 증가하는 방향"을 가리키는 벡터다. 연쇄법칙은 합성 함수의 미분을 곱으로 쪼갠다. 이 셋이 경사하강법·역전파, 즉 머신러닝 학습의 수학적 엔진이다.

## 왜 필요한가

- 미분이 실제로 뭘 재나 (변화율)
- 기울기가 왜 "방향"인가
- 연쇄법칙이 왜 역전파의 핵심인가 (ai-ml/[[neural-networks]])

## 미분 = 순간 변화율

```
f'(x) = lim(h→0) [f(x+h) - f(x)] / h
      = 그 점에서의 접선 기울기 (얼마나 가파른가)
```

- x를 조금 바꾸면 f가 얼마나 바뀌나 (민감도)
- 부호: 양수면 증가 중, 음수면 감소 중, 0이면 극점(최소/최대 후보)

### 코드로 확인: 수치 미분 = 해석 미분

`f(x)=x²`의 도함수는 `2x`. 수치적으로(중앙 차분) 확인:

```python
def numerical_deriv(f, x, h=1e-6):
    return (f(x+h) - f(x-h)) / (2*h)   # 정의를 그대로
```

실행:
```
x=1.0: 수치=2.000000  해석(2x)=2.000000
x=3.0: 수치=6.000000  해석(2x)=6.000000
x=5.0: 수치=10.000000  해석(2x)=10.000000
```

- 정의(극한)를 작은 h로 근사한 값이 해석해 `2x`와 일치
- **수치 미분**은 gradient checking(역전파 구현 검증)에 실제로 쓰임

## 연쇄법칙 (chain rule, 핵심)

**합성 함수의 미분 = 바깥 미분 × 안쪽 미분**:

```
g(x) = f(u(x))  →  g'(x) = f'(u(x)) · u'(x)
```

코드 확인 (`g(x)=sin(x²)`, `g'=cos(x²)·2x`):
```
연쇄법칙 g(x)=sin(x^2) at x=1.5: 수치=-1.884521  해석=-1.884521
```

- **왜 중요**: 신경망은 함수의 함수의 함수... (층층이 합성) → 미분이 각 층 미분의 **곱**
- 역전파([[neural-networks]])는 이 곱을 **출력→입력 방향으로** 효율적으로 계산하는 것
- 곱이 계속되므로 값이 1보다 작으면 소실, 크면 폭발 (ai-ml/[[training-dynamics]]의 기울기 문제 근원)

## 기울기 (gradient) = 가장 가파른 방향

다변수 함수의 미분 - **각 변수로의 편미분을 모은 벡터**:

```
f(x,y)  →  ∇f = [∂f/∂x, ∂f/∂y]
편미분 ∂f/∂x: 다른 변수 고정하고 x로만 미분
```

코드 확인 (`f(x,y)=x²+3y²`, `∇f=[2x, 6y]`):
```
grad f(2,1): 수치=[4.0000, 6.0000]  해석=[4.0000, 6.0000]
```

핵심 성질:
- **기울기 벡터는 함수가 가장 빠르게 증가하는 방향**을 가리킴
- 크기는 그 방향의 가파름
- 그래서 **반대 방향(-∇f)으로 가면 가장 빠르게 감소** → 경사하강법

## 경사하강법과의 연결

ai-ml/[[linear-models]]·[[neural-networks]]의 학습 엔진:

```
손실 L을 최소화 → -∇L 방향으로 파라미터 이동
w ← w - lr·∇L      (lr = 걸음 크기)
```

- 손실 함수의 기울기를 구해(역전파=연쇄법칙) 반대로 조금씩 이동
- **미분(변화율) + 기울기(방향) + 연쇄법칙(합성)** = ML 최적화의 전부
- 극점에서 ∇=0 (더 내려갈 곳 없음 - 수렴, 또는 안장점/국소최소)

## ML에 필요한 미적분 (범위)

CS/ML 관점에서 실제로 쓰는 것:
- **1계 미분·기울기**: 경사하강 (거의 전부 여기)
- **편미분·야코비안**: 다변수·벡터 함수
- **헤시안(2계)**: 곡률 - 뉴턴법·최적화 이론 (실무 딥러닝은 잘 안 씀, 비쌈)
- 적분보다 **미분이 압도적으로 중요** (학습 = 미분 기반 최적화)

## 셀프 체크

> [!question]- 미분 f'(x)는 정확히 무엇을 재는가?
> 그 점에서 함수의 순간 변화율, 즉 접선의 기울기다. x를 아주 조금 바꿨을 때 f가 얼마나 민감하게 바뀌는지를 나타낸다. 부호가 양수면 증가 중, 음수면 감소 중, 0이면 극점(최소/최대) 후보다.

> [!question]- 기울기(gradient) ∇f가 왜 "방향"이고, 경사하강법은 왜 −∇f로 가는가?
> ∇f는 각 변수의 편미분을 모은 벡터로, 함수가 가장 빠르게 증가하는 방향을 가리키고 크기는 그 가파름이다. 반대 방향 −∇f는 가장 빠르게 감소하는 방향이므로, 손실을 줄이려면 그쪽으로 파라미터를 이동한다. 그래서 갱신식이 w ← w − lr·∇L이 된다.

> [!question]- 연쇄법칙이 왜 역전파의 핵심인가?
> 신경망은 함수의 함수의 함수... 처럼 층층이 합성된 구조라, 전체 미분이 각 층 미분의 곱으로 분해된다(g'(x)=f'(u(x))·u'(x)). 역전파는 이 곱을 출력에서 입력 방향으로 효율적으로 계산하는 절차다. 곱이 누적되므로 값이 1보다 작으면 기울기 소실, 크면 폭발이 생긴다.

> [!question]- 수치 미분은 무엇이고 실무에서 어디에 쓰이나?
> 미분의 정의(극한)를 작은 h로 근사하는 것으로, 중앙 차분 (f(x+h)−f(x−h))/(2h)를 쓴다. 해석적 도함수와 값이 일치하는지 비교할 수 있어서, 역전파 구현이 맞는지 검증하는 gradient checking에 실제로 쓰인다.

## 연습문제

> [!example]- 계산하라: f(x,y)=x²+3y²의 기울기 ∇f를 구하고 점 (2,1)에서 값을 구하라
> **풀이**
> 편미분: ∂f/∂x=2x (y 고정), ∂f/∂y=6y (x 고정).
> 따라서 ∇f=[2x, 6y].
> (2,1)에서 ∇f=[2·2, 6·1]=[4, 6].
> 이 벡터가 그 점에서 함수가 가장 빠르게 증가하는 방향이고, 경사하강은 반대인 [−4, −6] 방향으로 이동한다.

> [!example]- 계산하라: g(x)=sin(x²)의 도함수를 연쇄법칙으로 구하고 x=1.5에서 값을 계산하라
> **풀이**
> u(x)=x²로 두면 g=sin(u). 바깥 미분 cos(u)·안쪽 미분 u'=2x.
> g'(x)=cos(x²)·2x.
> x=1.5: x²=2.25, cos(2.25)≈−0.6282, 2x=3.0.
> g'(1.5)≈−0.6282·3.0≈−1.8845. 노트의 수치 미분 결과 −1.884521과 일치한다.

> [!example]- 증명하라: 극소점이 아닌데도 ∇f=0이 될 수 있음을 반례로 보여라 (안장점)
> **풀이**
> f(x,y)=x²−y²를 보자. ∇f=[2x, −2y]이므로 원점 (0,0)에서 ∇f=[0,0]이다.
> 그러나 x축 방향(y=0)으로는 f=x²이라 원점이 최소지만, y축 방향(x=0)으로는 f=−y²이라 원점이 최대다.
> 따라서 원점은 극소도 극대도 아닌 안장점이다. ∇=0은 극점의 필요조건일 뿐 충분조건이 아니며, 경사하강이 여기서 멈출 수 있음을 보여준다.
