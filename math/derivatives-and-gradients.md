# 미분과 기울기 (Derivatives and Gradients)

## 한 줄 요약

미분은 "그 점에서 함수가 얼마나 가파른가"(순간 변화율)이고, 다변수로 확장한 기울기(gradient)는 "가장 가파르게 증가하는 방향"을 가리키는 벡터다. 연쇄법칙은 합성 함수의 미분을 곱으로 쪼갠다. 이 셋이 경사하강법·역전파, 즉 머신러닝 학습의 수학적 엔진이다.

## 왜 필요한가

- 미분이 실제로 뭘 재나 (변화율)
- 기울기가 왜 "방향"인가
- 연쇄법칙이 왜 역전파의 핵심인가 (ai-ml/[[neural-networks]])

## 미분 = 순간 변화율

```
f'(x) = lim(h→0) [f(x+h) - f(x)] / h
      = 그 점에서의 접선 기울기 (얼마나 가파른가)
```

- x를 조금 바꾸면 f가 얼마나 바뀌나 (민감도)
- 부호: 양수면 증가 중, 음수면 감소 중, 0이면 극점(최소/최대 후보)

### 코드로 확인: 수치 미분 = 해석 미분

`f(x)=x²`의 도함수는 `2x`. 수치적으로(중앙 차분) 확인:

```python
def numerical_deriv(f, x, h=1e-6):
    return (f(x+h) - f(x-h)) / (2*h)   # 정의를 그대로
```

실행:
```
x=1.0: 수치=2.000000  해석(2x)=2.000000
x=3.0: 수치=6.000000  해석(2x)=6.000000
x=5.0: 수치=10.000000  해석(2x)=10.000000
```

- 정의(극한)를 작은 h로 근사한 값이 해석해 `2x`와 일치
- **수치 미분**은 gradient checking(역전파 구현 검증)에 실제로 쓰임

## 연쇄법칙 (chain rule, 핵심)

**합성 함수의 미분 = 바깥 미분 × 안쪽 미분**:

```
g(x) = f(u(x))  →  g'(x) = f'(u(x)) · u'(x)
```

코드 확인 (`g(x)=sin(x²)`, `g'=cos(x²)·2x`):
```
연쇄법칙 g(x)=sin(x^2) at x=1.5: 수치=-1.884521  해석=-1.884521
```

- **왜 중요**: 신경망은 함수의 함수의 함수... (층층이 합성) → 미분이 각 층 미분의 **곱**
- 역전파([[neural-networks]])는 이 곱을 **출력→입력 방향으로** 효율적으로 계산하는 것
- 곱이 계속되므로 값이 1보다 작으면 소실, 크면 폭발 (ai-ml/[[training-dynamics]]의 기울기 문제 근원)

## 기울기 (gradient) = 가장 가파른 방향

다변수 함수의 미분 - **각 변수로의 편미분을 모은 벡터**:

```
f(x,y)  →  ∇f = [∂f/∂x, ∂f/∂y]
편미분 ∂f/∂x: 다른 변수 고정하고 x로만 미분
```

코드 확인 (`f(x,y)=x²+3y²`, `∇f=[2x, 6y]`):
```
grad f(2,1): 수치=[4.0000, 6.0000]  해석=[4.0000, 6.0000]
```

핵심 성질:
- **기울기 벡터는 함수가 가장 빠르게 증가하는 방향**을 가리킴
- 크기는 그 방향의 가파름
- 그래서 **반대 방향(-∇f)으로 가면 가장 빠르게 감소** → 경사하강법

## 경사하강법과의 연결

ai-ml/[[linear-models]]·[[neural-networks]]의 학습 엔진:

```
손실 L을 최소화 → -∇L 방향으로 파라미터 이동
w ← w - lr·∇L      (lr = 걸음 크기)
```

- 손실 함수의 기울기를 구해(역전파=연쇄법칙) 반대로 조금씩 이동
- **미분(변화율) + 기울기(방향) + 연쇄법칙(합성)** = ML 최적화의 전부
- 극점에서 ∇=0 (더 내려갈 곳 없음 - 수렴, 또는 안장점/국소최소)

## ML에 필요한 미적분 (범위)

CS/ML 관점에서 실제로 쓰는 것:
- **1계 미분·기울기**: 경사하강 (거의 전부 여기)
- **편미분·야코비안**: 다변수·벡터 함수
- **헤시안(2계)**: 곡률 - 뉴턴법·최적화 이론 (실무 딥러닝은 잘 안 씀, 비쌈)
- 적분보다 **미분이 압도적으로 중요** (학습 = 미분 기반 최적화)

## 연결

- 역전파 = 연쇄법칙 → ai-ml/[[neural-networks]]

- 기울기 소실/폭발 (곱의 누적) → ai-ml/[[training-dynamics]]
