# 랜덤 변수와 분포 (Random Variables)

## 한 줄 요약

랜덤 결과에 수를 대응시킨 것이 랜덤 변수. 기댓값·분산으로 요약하고, 주요 분포(이항·기하·정규·포아송)가 흔한 상황을 모델링한다. 알고리즘 분석·성능 모델링의 도구.

## 왜 필요한가

- 확률을 수치로 다루기 (기댓값·분산)
- 흔한 상황의 표준 분포
- 알고리즘·시스템 성능 모델링

## 랜덤 변수 (random variable)

랜덤 결과 → 수 (함수 [[sets-relations-functions]]):

- **이산(discrete)**: 셀 수 있는 값 (주사위, 동전 횟수)
- **연속(continuous)**: 실수 범위 (지연시간, 키)
- 예: X = 동전 10번 중 앞면 수

## 기댓값과 분산

분포를 두 수로 요약:

### 기댓값 (expectation)

```
E[X] = Σ x·P(x)   (이산)
```

- 평균, 장기 평균값
- **선형성**: `E[X+Y] = E[X]+E[Y]` (독립 무관! 강력) → 복잡한 걸 쪼개 분석
- 알고리즘 평균 시간 (algorithms/[[asymptotic-analysis]])

### 분산 (variance)

```
Var[X] = E[(X - E[X])²]   (평균에서 얼마나 퍼졌나)
표준편차 = √Var
```

- 흩어짐·불확실성
- **꼬리 위험**: 평균만 보면 안 됨 (p99 지연 - algorithms/[[selection]], distributed-systems/[[observability-basics]])

## 주요 분포

흔한 상황의 표준 모델:

### 이항 분포 (binomial)

**n번 독립 시행, 각 성공 확률 p** → 성공 횟수:
- 예: 동전 n번 중 앞면 수, 패킷 n개 중 손실 수
- E[X] = np

### 기하 분포 (geometric)

**첫 성공까지 시행 횟수**:
- 예: 재시도 몇 번에 성공 (network/[[tcp-reliability]]의 재전송)
- 무기억성(memoryless)

### 포아송 분포 (Poisson)

**단위 시간당 사건 수** (드문 사건):
- 예: 초당 요청 수, 시간당 장애 수
- 대기열 이론(queuing)의 기반 → 서버 부하 모델링 (network/[[internet-overview]]의 큐잉 지연)

### 정규 분포 (normal/Gaussian)

**종 모양, 많은 것의 합** → 중심극한정리:
- 자연·측정 오차가 정규 (computer-architecture/[[floating-point]] 노이즈)
- **중심극한정리(CLT)**: 독립 랜덤 변수 많이 합치면 → 정규에 근사 (원 분포 무관!) → 통계의 기반 [[statistics-basics]]
- 68-95-99.7 규칙 (표준편차 1/2/3 안에)

### 균등 분포 (uniform)

모든 값이 같은 확률:
- 랜덤 생성기, 해시(이상적, data-structures/[[hash-tables]]), 버킷 정렬 가정 (algorithms/[[linear-sorts]])

## 큰 수의 법칙 (LLN)

시행을 많이 하면 **표본 평균 → 기댓값**:
- 생일 역설 시뮬레이션([[probability-basics]])이 이론값에 수렴한 이유
- 몬테카를로 방법의 기반 (랜덤 샘플로 근사)

## CS 응용

- **알고리즘 평균 분석**: 기댓값 (algorithms/[[asymptotic-analysis]]) - 퀵정렬, 해시
- **성능 모델링**: 포아송 도착 + 대기열 (지연 예측)
- **꼬리 지연**: 분산·p99 (distributed-systems/[[observability-basics]])
- **랜덤화**: 스킵 리스트(data-structures/[[skip-list]]), 랜덤 피벗
- **ML**: 확률 모델, 샘플링 (ai-ml/)
- **집중 부등식**: 성능 보장 → [[concentration]]

## 셀프 체크

> [!question]- 기댓값과 분산은 각각 무엇을 요약하나? 공식은?
> 기댓값 `E[X] = Σ x·P(x)`는 랜덤 변수의 장기 평균값(중심)을 요약한다. 분산 `Var[X] = E[(X − E[X])²]`는 값이 평균에서 얼마나 퍼졌는지(흩어짐·불확실성)를 나타내며, 표준편차는 그 제곱근이다. 평균만 보면 꼬리 위험(p99 지연)을 놓치므로 분산이 중요하다.

> [!question]- 이항·기하·포아송·정규 분포는 각각 어떤 상황을 모델링하나?
> 이항은 n번 독립 시행의 성공 횟수(동전 앞면 수, 패킷 손실 수), 기하는 첫 성공까지의 시행 횟수(재전송 횟수), 포아송은 단위 시간당 드문 사건 수(초당 요청, 큐잉의 도착), 정규는 많은 독립 요인의 합(측정 오차)이다. 상황의 구조를 보고 알맞은 표준 분포를 고르는 게 모델링의 출발점이다.

> [!question]- 중심극한정리(CLT)가 말하는 것은? 왜 통계의 기반인가?
> 독립인 랜덤 변수를 많이 합치면 그 합(또는 평균)의 분포가 원래 분포가 무엇이든 정규 분포에 근사한다는 정리다. 덕분에 원 분포를 몰라도 표본 평균의 분포를 정규로 다룰 수 있어 신뢰구간·가설검정이 가능해진다. 68-95-99.7 규칙도 여기서 나온다.

> [!question]- 큰 수의 법칙(LLN)과 중심극한정리(CLT)는 무엇이 다른가?
> LLN은 시행을 많이 하면 표본 평균이 기댓값으로 수렴한다는(값이 어디로 가는지) 법칙이고, CLT는 그 표본 평균이 기댓값 주위로 어떤 모양(정규)으로 퍼지는지를 말한다. LLN은 몬테카를로가 정답에 수렴함을 보장하고, CLT는 그 오차의 분포를 알려준다. 하나는 수렴점, 하나는 수렴의 형태를 다룬다.

## 연습문제

> [!example]- 계산하라: 공정한 주사위(1~6) 한 번의 기댓값 E[X]와 분산 Var[X].
> **풀이**
> E[X] = (1+2+3+4+5+6)/6 = 21/6 = 3.5.
> E[X²] = (1+4+9+16+25+36)/6 = 91/6 ≈ 15.1667.
> Var[X] = E[X²] − (E[X])² = 91/6 − (3.5)² = 15.1667 − 12.25 = 2.9167 (= 35/12).
> 표준편차 = √(35/12) ≈ 1.708.

> [!example]- 증명하라: 성공 확률 p인 독립 시행 n번의 성공 횟수 X에 대해 E[X] = np.
> **풀이**
> X를 각 시행의 지시 변수 합으로 쓴다: X = X₁ + X₂ + ... + Xₙ, 여기서 Xᵢ는 i번째가 성공이면 1, 아니면 0.
> 각 지시 변수의 기댓값은 E[Xᵢ] = 1·p + 0·(1−p) = p.
> 기댓값의 선형성(독립 무관)에 의해 E[X] = Σ E[Xᵢ] = n·p = np.
> 이처럼 지시 변수 + 선형성으로 이항 기댓값을 곱셈 하나 없이 얻는다.

> [!example]- 반례를 찾아라: "E[X+Y] = E[X]+E[Y]는 X, Y가 독립일 때만 성립한다"는 주장이 틀림을 보여라.
> **풀이**
> 반례로 Y = X인 완전 종속 경우를 잡는다. X가 공정한 동전으로 0 또는 1 (각 1/2)이라 하자.
> E[X] = 0.5, E[Y] = E[X] = 0.5. X+Y = 2X이므로 E[X+Y] = E[2X] = 2·0.5 = 1.0.
> E[X]+E[Y] = 0.5+0.5 = 1.0으로 일치한다.
> X와 Y는 완전히 종속인데도 선형성이 성립하므로, 선형성에 독립이 필요하다는 주장은 거짓이다. (독립이 필요한 것은 Var[X+Y]=Var[X]+Var[Y] 쪽이다)
