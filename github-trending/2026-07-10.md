오늘 트렌딩은 "AI 에이전트 스킬(skills)" 생태계가 완전히 상위권을 뒤덮은 하루였습니다. Claude Code의 `.claude` 디렉터리 기반 스킬 공유 저장소 두 개가 나란히 1,000개 이상의 스타를 얻었고, 에이전트용 프로덕션 스킬 모음, MCP 서버, AI 에이전트 장기 기억 저장소까지 이어지며 "에이전트에게 어떤 지식·행동 패턴을 파일로 넘겨줄 것인가"라는 주제가 압도적입니다. 그 사이사이에 Bun, TypeScript, Catch2, asio, abseil-cpp, yaml-cpp처럼 오랫동안 실무에서 쓰여온 시스템 프로그래밍/인프라 라이브러리들도 나란히 트렌딩에 올라, 에이전트 열풍과 무관하게 꾸준히 참조되는 기본기 프로젝트들의 존재감도 확인할 수 있는 하루였습니다.

### librats/librats

- <https://github.com/librats/librats>
- 언어: C++
- 스타: 140
- 생성일: 2025-07-11

Noise_XX 프로토콜(Curve25519 키 교환 + ChaCha20-Poly1305)로 암호화된 TCP 전송을 기본으로 하는 C++17 P2P 네트워킹 라이브러리입니다. 최소한의 코어 Node 위에 DHT 탐색, pub/sub 메시징, 파일 전송 같은 기능을 옵션 서브시스템으로 얹는 모듈형 구조를 택했고, 스레드 간 락 없이 리액터별로 연결 상태를 분산 처리하는 shared-nothing 리액터 트랜스포트가 특징입니다. epoll/kqueue/IOCP를 플랫폼별로 활용하고 BitTorrent Mainline DHT 호환, UPnP/NAT-PMP를 통한 NAT 트래버설까지 지원하며, GoogleTest 기반 테스트와 ASan/TSan 지원, libp2p JS 대비 메모리 사용량을 31~50배 줄였다는 벤치마크까지 공개해 완성도를 뒷받침합니다.
