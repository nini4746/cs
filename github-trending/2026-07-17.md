## PostHog/posthog

- <https://github.com/PostHog/posthog>
- 언어: Python
- 오늘 스타: 437 (총 36,075)

제품 분석, 세션 리플레이, 기능 플래그, 에러 추적, A/B 테스트, LLM 애플리케이션을 위한 AI observability까지 한 플랫폼에 묶은 오픈소스 도구입니다. Python·TypeScript·Rust로 이루어진 모노레포 구조에 25개가 넘는 데이터 파이프라인 연동을 갖추고 있고, 핵심 코드는 MIT 라이선스이되 일부 기능은 `ee` 디렉터리에 별도 라이선스로 분리해 오픈소스와 상업 모델을 병행하는 구조가 특징입니다. "에이전트가 문제를 진단하고 수정까지 할 수 있도록 컨텍스트를 모아준다"는 최근 지향점을 보면, 관측 가능성 도구 자체도 AI 에이전트를 염두에 두고 재설계되는 흐름을 보여주는 사례입니다.
