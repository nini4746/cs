### clojurewasm/zwasm

- <https://github.com/clojurewasm/zwasm>
- 언어: Zig
- 오늘 스타: 149 (생성일: 2026-02-10)

Zig로 완전히 새로 작성한 WebAssembly 런타임입니다. 인터프리터, ARM64/x86_64용 JIT, `.cwasm` 산출물을 만드는 AOT 컴파일까지 세 가지 실행 경로를 모두 구현했고, WebAssembly 1.0/2.0에 대해 스펙 테스트스위트를 macOS·Linux·Windows 세 플랫폼에서 100% 통과시킨다고 밝히고 있습니다. 인터프리터와 JIT의 실행 결과를 비교하는 differential testing과, use-after-free를 검증한 보수적 스택 스캔 GC까지 갖춰, 런타임 하나를 밑바닥부터 구현하며 스펙 준수와 검증을 어떻게 쌓아올리는지 볼 수 있는 자료입니다.

### prisma-risk/tsoracle

- <https://github.com/prisma-risk/tsoracle>
- 언어: Rust
- 오늘 스타: 55 (생성일: 2026-05-19)

gRPC로 통신하며 클러스터 전체에서 엄격히 단조 증가하는 타임스탬프와 빈틈없는(gapless) ID 시퀀스를 발급하는 분산 타임스탬프 오라클입니다. 합의 계층을 OpenRaft·OmniPaxos 중 고를 수 있게 trait로 추상화했고, 크래시 후에도 시간이 되돌아가지 않도록 윈도우 상태를 fsync하는 점이 눈에 띕니다. postcard 디코더에 대한 커버리지 유도 퍼징과 failpoint 기반 크래시 테스트, OpenSSF Scorecard 배지까지 갖추고 있어, 분산 시스템의 시간 문제를 다루는 코드베이스가 어느 수준까지 검증되어야 하는지 참고하기 좋습니다.

### BenChaliah/Tensa-Lang

- <https://github.com/BenChaliah/Tensa-Lang>
- 언어: C++
- 오늘 스타: 77 (생성일: 2026-02-04)

LLM 추론 엔진의 연산(어텐션, 정규화, 샘플링)을 직접 작성할 수 있는 텐서 중심 언어이자 컴파일러입니다. `.tl` 소스를 S-expression IR로 변환한 뒤 MLIR의 linalg/scf/arith 다이얼렉트로 낮추고, CPU 경로는 LLVM JIT으로, CUDA 경로는 NVVM을 거쳐 cubin으로 컴파일해 실행하는 파이프라인을 갖췄습니다. 스케줄링(타일링, 병렬화, 메모리 배치)을 언어 차원의 `with` 절로 표현하게 만든 설계와, 실제로 Llama2 7B·Qwen2.5 0.5B를 구동하는 예제까지 있다는 점에서, MLIR을 이용해 도메인 특화 컴파일러를 여러 하드웨어 타깃으로 확장하는 방법을 구체적으로 볼 수 있습니다.
