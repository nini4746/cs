Rust의 MIR을 C 소스로 트랜스파일한 뒤 시스템 C 컴파일러(gcc, clang)로 오브젝트 파일을 만드는 실험적 rustc 코드젠 백엔드입니다. rustc에 대체 백엔드로 꽂아 넣을 수 있는 형태로 통합되어 있고, 이 백엔드로 빌드한 컴파일러가 rustc UI 테스트 스위트의 99.2%를 통과한다고 밝히고 있어 MIR 수준에서 Rust 시맨틱을 상당히 충실히 보존하고 있음을 보여줍니다.
