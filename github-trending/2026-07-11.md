## wonderwhy-er/DesktopCommanderMCP

- <https://github.com/wonderwhy-er/DesktopCommanderMCP>
- 언어: TypeScript
- 오늘 스타: 900

Claude 같은 LLM에게 터미널 명령 실행, 파일 시스템 검색, diff 기반 파일 편집 기능을 제공하는 MCP(Model Context Protocol) 서버입니다. 로컬 시스템 조작(명령 실행·파일 읽기·검색)을 MCP 툴로 노출해 에이전트가 개발자 컴퓨터를 직접 다루게 하는 구조로, 오늘 트렌딩 전체에서 가장 큰 폭으로 스타가 늘어난 저장소입니다. MCP 서버를 직접 구현해보고 싶을 때 참고할 만한 실전 예시입니다.
