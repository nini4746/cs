## Dicklesworthstone/destructive_command_guard

- <https://github.com/Dicklesworthstone/destructive_command_guard>
- 언어: Rust
- 오늘 스타: 444 (총 2,501)

AI 에이전트가 실행하려는 위험한 git·셸 명령을 사전에 차단하는 가드레일 도구입니다. 에이전트가 `rm -rf`, `git reset --hard`, force push 같은 되돌리기 어려운 명령을 실행하기 전에 가로채 차단하는 구조로 보이며, Rust로 작성돼 있어 별도 런타임 의존 없이 가볍게 셸 앞단에 끼워 넣을 수 있다는 점이 특징입니다. Claude Code 같은 코딩 에이전트가 실제 개발 환경에서 명령을 자율적으로 실행하는 사례가 늘면서, "에이전트가 실수로 저장소를 날리는" 상황을 막는 방어 계층에 대한 수요가 하루 만에 444개의 스타로 나타난 것으로 보입니다. 에이전트 안전장치를 설계할 때 어떤 명령 패턴을 화이트/블랙리스트로 다뤄야 하는지 참고하기 좋은 사례입니다.
