### MazuNIX/mazu

- <https://github.com/MazuNIX/mazu>
- 언어: C
- 스타: 116 (2026-05-09 생성)

RISC-V 64비트 베어메탈 위에서 동작하는 하드 리얼타임 OS로, Linux의 모듈형 서브시스템·우선순위 상속 뮤텍스·per-CPU 구조와 Plan 9의 "모든 것이 파일" 철학(`/dev`, `/proc`, `/net`을 읽을 수 있는 합성 파일시스템으로 노출)을 의도적으로 조합했습니다. EDF 승인 제어를 포함한 하드 데드라인 스케줄링, SACK과 커넥션 풀링을 갖춘 TCP/IP 스택을 프리엠프터블 태스크로 통합했으며, 반대로 로더블 모듈이나 태스크 간 가상 메모리 격리, VFS 페이지 캐시, 소켓 API는 의도적으로 배제했다고 명시합니다. lockdep 검증, 스케줄러 불변식 체크, UBSan 트랩 모드 같은 검증 도구와 PSE51 호환성 매트릭스까지 문서화되어 있어, "무엇을 뺐는지"까지 근거를 대는 설계 문서 수준이 높다고 판단했습니다.
