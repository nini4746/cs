# 펌핑 보조정리 (Pumping Lemma)

## 한 줄 요약

정규 언어라면 충분히 긴 문자열은 "펌핑"(반복)해도 여전히 그 언어에 속한다. 이 성질을 위반하는 언어는 정규가 아니다. aⁿbⁿ이 정규가 아님을 증명하는 도구.

## 왜 필요한가

- "이 언어는 정규식/DFA로 표현 불가"를 증명하는 법
- 유한 오토마타의 근본 한계
- 왜 균형 괄호를 정규식으로 못 하나

## 직관: 유한 상태의 한계

DFA는 **유한한 상태**만 가짐 ([[dfa-nfa]]). 상태가 p개인 DFA에 길이 ≥ p인 문자열을 넣으면:

- 비둘기집 원리(math/[[combinatorics]]): p+1개 상태를 방문하는데 상태는 p개 → **어떤 상태를 두 번 방문**
- 두 번 방문 = **사이클** → 그 사이클 부분을 반복(펌핑)해도 DFA는 여전히 수락

이 "반복해도 수락"이 정규 언어의 필연적 성질. 이걸 어기면 정규 아님.

## 펌핑 보조정리 (형식)

L이 정규 언어면, **어떤 펌핑 길이 p가 존재**해서 **L에 속하는** 길이 ≥ p인 모든 문자열 s(즉 s ∈ L, |s| ≥ p)를 s = xyz로 나눌 수 있고:

1. **|y| > 0** (y는 비어있지 않음 - 펌핑할 게 있음)
2. **|xy| ≤ p** (y는 앞쪽 p글자 안)
3. **모든 i ≥ 0에 대해 xyⁱz ∈ L** (y를 0번 이상 반복해도 L에 속함)

y가 사이클 부분. 이걸 반복(i번)해도 언어에 남아야 함.

## 사용법: 비정규성 증명 (반증)

**대우로 증명**: L이 위 성질을 위반하면 정규가 아님. 게임 형식:

1. L이 정규라고 **가정** (모순 유도)
2. 펌핑 길이 p가 존재 (상대가 줌, 값은 모름)
3. **L에 속하는** 길이 ≥ p인 **영리한 문자열 s를 선택** (s ∈ L, 내가 고름)
4. 상대의 모든 분할 xyz에 대해 (조건 1,2 만족하는)
5. **어떤 i에서 xyⁱz ∉ L**임을 보임 → 조건 3 위반 → 모순
6. → L은 정규가 아님

## 예: aⁿbⁿ이 정규 아님

L = {aⁿbⁿ : n ≥ 0} (a가 n개 뒤 b가 n개, 개수 같음):

1. 정규라 가정, 펌핑 길이 p
2. **s = aᵖbᵖ 선택** (길이 2p ≥ p)
3. 조건 2(|xy| ≤ p)에 의해 xy는 앞쪽 p글자 = **전부 a**. 따라서 y = a^k (k>0)
4. **i=2로 펌핑**: xy²z = a^(p+k)b^p → a가 b보다 많음
5. a 개수 ≠ b 개수 → aⁿbⁿ 형태 아님 → ∉ L → 모순
6. → **L은 정규가 아님** ∎

직관: DFA는 유한 상태라 "지금까지 a를 몇 개 봤나"를 무한히 셀 수 없음. n이 커지면 못 셈.

## 무엇이 정규가 아닌가

펌핑 보조정리로 증명되는 비정규 언어들 - 공통점은 **무한히 세거나 기억해야** 함:

- **aⁿbⁿ**: 개수 맞추기
- **균형 괄호**: 여는/닫는 짝 (중첩 깊이 셈) → 프로그래밍 언어가 정규가 아닌 이유
- **회문(palindrome)**: 앞뒤 대칭 기억
- **aⁿbⁿcⁿ**: 더 강함 (문맥 자유도 아님 → [[cfl-pumping]])

이들은 문맥 자유 문법([[context-free-grammars]])이나 그 이상이 필요.

## 주의: 필요조건이지 충분조건 아님

- 펌핑 보조정리 **위반 → 비정규** (증명 도구로 유효)
- 펌핑 성질 **만족 ≠ 정규** (일부 비정규 언어도 만족). 정규성을 **증명**하는 데는 못 씀
- 정규 증명은 DFA/정규식을 직접 제시 ([[dfa-nfa]])

Myhill-Nerode 정리가 더 강력한 판별법 (필요충분) → [[dfa-minimization]].

## 셀프 체크

> [!question]- 펌핑 보조정리의 세 조건(|y|>0, |xy|≤p, 모든 i≥0에 xyⁱz∈L)은 각각 무엇을 보장하는가?
> |y|>0은 펌핑할 게 실제로 존재함을, |xy|≤p는 y가 앞쪽 p글자 안(즉 첫 사이클)에 있음을, xyⁱz∈L은 사이클 부분 y를 0번 이상 반복해도 언어에 남아야 함을 보장한다.

> [!question]- 비정규성 증명 게임에서 무엇을 상대가 고르고 무엇을 내가 고르는가?
> 상대(정규라는 가정)가 펌핑 길이 p와 분할 xyz를 준다(단 조건 1,2 만족). 나는 길이 ≥ p인 영리한 문자열 s와 펌핑 횟수 i를 고른다. 상대의 모든 유효 분할에 대해 어떤 i에서 xyⁱz∉L을 보이면 모순이다.

> [!question]- s=aᵖbᵖ에서 왜 y가 반드시 전부 a로만 이루어지는가?
> 조건 2에서 |xy|≤p인데 s의 앞쪽 p글자는 전부 a이므로 xy 전체가 a 구간 안에 있다. 따라서 그 부분집합인 y도 a^k (k>0) 꼴이다.

> [!question]- 펌핑 성질을 만족하면 정규 언어인가?
> 아니다. 펌핑은 필요조건일 뿐이라 일부 비정규 언어도 성질을 만족한다. 위반하면 비정규임을 증명할 수 있지만, 만족한다고 정규를 증명하지는 못한다. 정규 증명은 DFA/정규식을 직접 제시하거나 Myhill-Nerode를 쓴다.

## 연습문제

> [!example]- 문제: L = {aⁿbᵐ : n > m ≥ 0}이 정규가 아님을 펌핑 보조정리로 증명하라.
> **풀이**
> 1. L이 정규라 가정, 펌핑 길이 p.
> 2. s = a^(p+1)b^p 선택 (n=p+1 > m=p이므로 s∈L, 길이 ≥ p).
> 3. 조건 2에 의해 xy는 앞쪽 p글자 = 전부 a → y = a^k (k>0).
> 4. i=0으로 펌핑(y 제거): xy⁰z = a^(p+1-k)b^p. 이제 a 개수 = p+1-k ≤ p = b 개수이므로 n > m이 깨짐.
> 5. → a^(p+1-k)b^p ∉ L → 모순. 따라서 L은 비정규. ∎

> [!example]- 문제: L = {a^(n²) : n ≥ 0} (완전제곱 개수의 a)가 정규가 아님을 증명하라.
> **풀이**
> 1. 정규라 가정, 펌핑 길이 p.
> 2. s = a^(p²) 선택 (길이 p² ≥ p).
> 3. 분할 xyz에서 |y|=k, 1≤k≤p (조건 1,2).
> 4. i=2로 펌핑: |xy²z| = p²+k. 그런데 p² < p²+k ≤ p²+p < (p+1)² = p²+2p+1. 즉 xy²z의 길이가 연속한 두 제곱수 p²과 (p+1)² 사이에 끼어 완전제곱이 아니다.
> 5. → xy²z ∉ L → 모순. 따라서 L은 비정규. ∎

## 파인만

> [!note]- 백지에 비정규성 증명의 6단계 게임을 적고, aⁿbⁿ에 직접 적용해 처음부터 끝까지 써보라. 막히면 그 단계만 다시.
> **점검 포인트**: (1) 세 조건 각각의 역할을 말할 수 있는가, (2) "상대가 p·분할, 내가 s·i"라는 게임 구도를 재현할 수 있는가, (3) 조건 |xy|≤p를 이용해 y의 구성을 제약하는 논리를 설명할 수 있는가.
