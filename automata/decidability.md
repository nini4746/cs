# 결정 가능성 (Decidability)

## 한 줄 요약

어떤 문제는 알고리즘으로 절대 풀 수 없다. 정지 문제가 대표 - "이 프로그램이 멈추는가"를 판정하는 프로그램은 존재할 수 없다. 대각선 논법으로 증명된다.

## 왜 필요한가

- 계산에 원리적 한계가 있다는 충격적 사실
- 왜 완벽한 버그 검출기/바이러스 검출기가 불가능한가
- 컴퓨터 과학의 가장 깊은 결과 중 하나

## 결정 가능 vs 인식 가능

튜링 머신([[turing-machines]])으로 정의:

- **결정 가능(decidable)**: TM이 **항상 정지**하며 예/아니오. 완전한 알고리즘
- **인식 가능(recognizable, r.e.)**: 예인 경우는 정지해 수락, **아니오면 영원히 안 멈출 수도**
- **결정 불가**: 어떤 TM도 항상 정지하며 못 푸는 문제

핵심 질문: 모든 문제가 결정 가능한가? **아니다.**

## 정지 문제 (Halting Problem)

**"프로그램 P와 입력 x가 주어졌을 때, P(x)가 정지하는가?"**

이걸 판정하는 알고리즘 HALT(P, x)가 존재할 것 같지만 - **존재할 수 없음**. 튜링의 증명 (1936):

### 대각선 논법 증명

1. HALT(P, x)가 존재한다고 **가정** (P가 x에서 정지하면 true)
2. 이걸로 이상한 프로그램 D를 만듦:

```
D(P):
   if HALT(P, P) == true:   # P가 자기 자신 입력에 정지하면
       무한 루프            # D는 안 멈춤
   else:
       정지
```

3. **D(D)를 실행**하면?
   - D(D)가 정지한다면 → HALT(D,D)=true → D는 무한 루프 → **정지 안 함** (모순)
   - D(D)가 안 멈춘다면 → HALT(D,D)=false → D는 정지 → **정지함** (모순)
4. 어느 쪽도 모순 → **HALT는 존재할 수 없음** ∎

자기 참조(D가 자신을 입력으로)가 모순을 만듦 - 칸토어의 대각선 논법([[sets-relations-functions]])과 같은 구조. "거짓말쟁이 역설"의 계산 버전.

## 정지 문제의 함의

정지 문제가 결정 불가 → 많은 실용 문제도 결정 불가 (환원 → [[reductions]]):

- **완벽한 버그 검출**: "이 프로그램이 크래시하나"를 항상 판정 불가
- **완벽한 바이러스 검출**: "이 코드가 악성인가" 일반적으로 불가
- **코드 최적화 한계**: "이 두 프로그램이 같은가" 불가
- **완벽한 데드락 검출**: 일반적으로 불가

그래서 실무 도구는 **근사**: 보수적(거짓 양성 허용), 특정 케이스만, 휴리스틱. 완벽은 원리적으로 불가능.

## Rice 정리

정지 문제를 일반화한 강력한 결과:

**"프로그램의 의미적 성질(무엇을 계산하나) 중 자명하지 않은 것은 전부 결정 불가."**

- 자명하지 않음 = 어떤 프로그램은 만족, 어떤 건 안 함
- 예: "이 프로그램이 항상 0을 출력하나", "이 함수가 상수 함수인가" 등 전부 결정 불가
- 함의: 프로그램의 **행동**에 관한 흥미로운 질문은 대개 결정 불가. 구문(syntax)은 판정 가능하지만 의미(semantics)는 아님

## 결정 가능한 것들 (대조)

전부 불가능한 건 아님. 결정 가능한 문제:

- 정규 언어 소속 (DFA 실행 → [[dfa-nfa]])
- CFL 소속 (CYK 파싱 → [[cfg-to-parsing]])
- 소수 판정, 정렬 검증 등 대부분 일상 알고리즘
- 유한한 탐색으로 끝나는 것

경계: **유한 시간 보장**이 있으면 결정 가능. 무한 탐색 가능성(정지 여부, 프로그램 의미)이면 위험.

## 계층 정리

```
결정 가능 ⊊ 인식 가능(r.e.) ⊊ 모든 언어
정렬 등     정지 문제(수락측)   대부분(비가산 무한)
```

- 정지 문제는 인식 가능(멈추면 수락)하지만 결정 불가
- 정지 문제의 **여집합**은 인식조차 불가
- 대부분의 언어는 인식조차 불가 (언어는 비가산, TM은 가산 → 셀 수 없이 많은 언어가 어떤 TM에도 대응 안 됨)
