# CS 공부 노트

배운 것, 배울 것 전부 여기에 정리.

## 구조

- `os/` - 운영체제 (프로세스, 스레드, 메모리 관리, 스케줄링, 동기화)
- `network/` - 네트워크 (TCP/IP, HTTP, DNS, 소켓)
- `algorithms/` - 알고리즘 (정렬, 탐색, DP, 그래프)
- `data-structures/` - 자료구조 (배열, 리스트, 트리, 해시, 힙)
- `database/` - 데이터베이스 (SQL, 인덱스, 트랜잭션, 정규화)
- `computer-architecture/` - 컴퓨터 구조 (CPU, 캐시, 메모리 계층)
- `programming-languages/` - 프로그래밍 언어 (언어별 개념, 패러다임, 컴파일러/인터프리터)
- `web/` - 웹 (브라우저 동작, 렌더링, REST, 인증)
- `security/` - 보안 (암호화, 해싱, 취약점)
- `distributed-systems/` - 분산 시스템 (CAP, 합의, 메시지 큐, 캐싱 전략)

## 규칙

- 파일명: kebab-case, 주제 단위 (예: `os/process-vs-thread.md`)
- 한 파일 = 한 주제
