# 엔트로피와 정보 (Entropy and Information)

## 한 줄 요약

정보량은 "놀람의 정도"다 - 드문 사건일수록 정보가 많다(`-log p`). 엔트로피는 그 정보량의 기댓값, 즉 분포의 **평균 불확실성**이자 그 분포를 무손실로 표현하는 데 필요한 **평균 비트 수의 하한**이다. 균등 분포에서 최대, 결정적 분포에서 0.

## 왜 필요한가

- "정보"를 어떻게 수치로 재나
- 왜 하필 `-log p`인가 (자기정보)
- 엔트로피가 압축·ML과 무슨 상관인가

## 자기정보 (self-information)

한 사건의 정보량 - **얼마나 놀라운가**:

```
I(x) = -log₂ p(x)   (단위: 비트)
```

왜 이 형태여야 하나 (공리로부터 유도):
- **드물수록 정보 많다**: p 작으면 I 큼 (확실한 일엔 정보 없음)
- **확실한 사건은 정보 0**: p=1 → I=0 (놀람 없음)
- **독립 사건은 더해짐**: `I(x,y)=I(x)+I(y)` (독립이면 `p(x,y)=p(x)p(y)`, log가 곱을 합으로) → **log만 이 성질 만족**
- 밑이 2면 비트, e면 nat, 10이면 dit

## 섀넌 엔트로피

자기정보의 **기댓값** - 분포 전체의 평균 불확실성:

```
H(X) = E[I(X)] = -Σ p(x) log₂ p(x)   (비트)
```

- "이 분포에서 결과 하나를 알려면 평균 몇 비트 필요한가"
- 불확실성·무질서·정보량의 척도 (같은 것의 세 얼굴)

## 코드로 확인

여러 분포의 엔트로피:

```python
def entropy(probs):
    return -sum(p*math.log2(p) for p in probs if p>0)
```

실행:
```
  H = 1.0000 bits  <- 공정한 동전 [0.5, 0.5]
  H = 0.4690 bits  <- 편향 동전 [0.9, 0.1]
  H = -0.0000 bits  <- 완전 편향 [1.0, 0.0]
  H = 2.5850 bits  <- 공정한 주사위 [1/6 x6]
  H = 2.0000 bits  <- 4글자 균등 [0.25 x4]
  H = 1.3568 bits  <- 4글자 편향 [0.7,0.1,0.1,0.1]
```

읽는 법:
- **공정한 동전 = 1비트**: 가장 불확실 (딱 예/아니오 하나 필요)
- **편향 동전(0.9/0.1) = 0.47비트**: 거의 앞면일 걸 아니까 덜 불확실 → 압축 여지
- **결정적(확률 1) = 0비트**: 결과가 뻔함 (정보 없음)
- **균등 4글자 = 2비트 = log₂4**: 균등이 그 크기에서 최대
- **편향 4글자 = 1.36비트**: 같은 4글자라도 편향되면 엔트로피↓ (Huffman이 노리는 것)

## 핵심 성질

- **범위**: `0 ≤ H(X) ≤ log₂|X|`
  - **최소 0**: 결정적 분포 (한 결과만 가능)
  - **최대 log₂n**: 균등 분포 (n개 결과가 다 똑같이 그럴듯 - 최대 불확실) → [[mutual-information]]의 KL로 증명
- **연속성·대칭성**: 확률을 재배열해도 H 불변 (라벨 무관)
- **가법성**: 독립 결합은 엔트로피 더해짐 → [[joint-conditional-entropy]]

## 왜 비트 수의 하한인가 (핵심 의미)

엔트로피의 진짜 의미 - **압축의 이론적 한계**:

```
분포 X를 무손실로 부호화할 때
평균 부호 길이 ≥ H(X)   (섀넌 소스 코딩 정리)
```

- 편향 동전(H=0.47)은 심볼당 평균 0.47비트까지 압축 가능 (1비트 아니라)
- 이게 [[source-coding-theorem]]과 [[huffman-and-entropy-coding]]의 토대
- "엔트로피 = 데이터에 실제로 든 정보량 = 압축 바닥"

## ML·확률과의 연결

- **cross-entropy 손실**: 예측 분포로 실제를 부호화하는 평균 비트 → 분류 학습의 손실 ([[information-theory-in-ml]], ai-ml/[[linear-models]])
- **베이즈·확률**: 조건부 확률·독립이 정보 정의의 바탕 (math/[[probability-basics]])
- **해시·랜덤**: 최대 엔트로피 = 최대 예측 불가 (security/[[hashing]]의 무작위성)

## 연결

- 결합·조건부로 확장 → [[joint-conditional-entropy]]
