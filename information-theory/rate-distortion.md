# 율-왜곡 이론 (Rate-Distortion)

## 한 줄 요약

손실 압축의 이론 - "허용 왜곡 D를 정하면, 그걸 지키는 최소 비트율 R(D)는 얼마인가". 무손실(엔트로피)이 절대 하한이라면, 왜곡을 허용하면 그 아래로 갈 수 있다. R(D)는 왜곡 제약 하의 최소 상호정보량이고, JPEG·MP3·손실 신경망 압축이 다 이 트레이드오프 위에 있다.

## 왜 필요한가

- 무손실 하한(엔트로피) 아래로 어떻게 가나
- 비트를 줄이면 품질이 얼마나 나빠지나 (정량적 트레이드오프)
- JPEG·MP3가 왜 "충분히 좋은가"

## 왜 손실 압축인가

```
무손실: 완벽 복원, 하지만 엔트로피 H가 하한 ([[source-coding-theorem]])
손실:   약간의 왜곡 허용 → H보다 훨씬 적은 비트
```

- 이미지·음성·영상은 **완벽 복원 불필요**: 사람이 못 느끼는 차이는 버려도 됨
- 연속 신호는 무한 정밀도 → 무손실 자체가 불가능 (반드시 양자화)
- 핵심 질문: **얼마나 버려야 얼마나 나빠지나**

## 왜곡 척도

"원본과 복원본이 얼마나 다른가"를 수치화:

```
D = E[ d(X, X̂) ]      (d = 왜곡 함수)
```

- **MSE**(제곱오차): 흔하지만 지각과 안 맞을 때 많음
- **지각 척도**: 사람 눈·귀 모델 (JPEG는 고주파에 둔감함을 이용)
- 왜곡 정의가 무엇을 "버려도 되는지" 결정

## 율-왜곡 함수 R(D)

**왜곡 D 이하를 지키는 최소 비트율**:

```
R(D) = min I(X; X̂)     단,  E[d(X,X̂)] ≤ D
          p(x̂|x)
```

- [[channel-capacity]]와 쌍대: 용량은 max I(제약: 전력), R(D)는 min I(제약: 왜곡)
- 성질:
  - **R(0) = H(X)** (무왜곡 = 무손실 하한, 이산의 경우)
  - **D 커질수록 R 감소** (많이 버릴수록 적은 비트) - 단조 감소·볼록
  - D가 최대(전부 버림)면 R=0

### 가우시안 예 (닫힌 형태)

분산 σ²인 가우시안 소스, MSE 왜곡:
```
R(D) = ½ log₂(σ²/D)   (0 ≤ D ≤ σ²)
```
- 왜곡 D를 절반으로 → 비트율 ½비트 증가 (역으로 1비트 추가 = 왜곡 4배 감소)
- **6 dB / 비트 법칙**: 1비트당 SNR 약 6dB (양자화 상식의 이론적 근거)

## 소스-채널 분리 (전체 그림)

정보이론 세 정리가 맞물림:

```
소스 → [소스 코딩] → [채널 코딩] → 채널
       R(D)로 압축     C 이하 부호     잡음
       (버림)          (중복 추가)
```

- **압축(R(D))은 중복 제거, 채널 부호(C)는 중복 추가** - 정반대인데 분리 최적 ([[channel-capacity]]의 분리 정리)
- 왜곡 허용 압축(여기) + 오류 정정([[error-correcting-codes]])이 독립 설계 가능

## 실전 손실 압축

R(D) 이론을 구현한 것들:

```
JPEG:  DCT 변환 → 고주파 계수 양자화(버림) → 엔트로피 코딩
MP3/AAC: 주파수 변환 → 심리음향 모델로 안 들리는 것 버림 → 엔트로피 코딩
H.264/265: 예측 + 변환 + 양자화 + 엔트로피 코딩
```

공통 구조:
1. **변환**: 에너지를 소수 계수에 집중 (DCT/wavelet, math/[[eigenvalues]]의 기저 변환)
2. **양자화**: 여기서 정보 버림 (R(D) 트레이드오프의 실체) - 되돌릴 수 없음
3. **엔트로피 코딩**: 남은 것 무손실 압축 ([[huffman-and-entropy-coding]])

- 품질 슬라이더 = D 조절 (양자화 세기)

## ML과의 연결

- **오토인코더·VAE**: 잠재 공간으로 압축 = 학습된 R(D) (병목이 비트 예산)
- **신경망 이미지 압축**: R(D)를 직접 손실로 최적화 (rate + λ·distortion)
- **information bottleneck**: 표현이 입력을 얼마나 압축하며 라벨 정보를 지키나 → [[information-theory-in-ml]]

## 셀프 체크

> [!question]- R(D) 함수의 정의와 R(0)의 값은?
> `R(D) = min_{p(x̂|x)} I(X;X̂)` subject to `E[d(X,X̂)] ≤ D`. 곧 왜곡 D 이하를 지키는 최소 비트율. 이산 소스에서 `R(0)=H(X)`로, 무왜곡이면 무손실 하한인 엔트로피와 같아진다.

> [!question]- 채널 용량 C와 율-왜곡 R(D)는 왜 "쌍대(dual)"인가?
> 둘 다 상호정보량의 극값 문제인데 방향이 반대다. 용량은 전력 제약 하 `max I`(중복 추가로 잡음 견딤), R(D)는 왜곡 제약 하 `min I`(중복 제거로 압축). 하나는 잡음을 이기려 정보를 늘리고 하나는 버려도 되는 정보를 줄인다.

> [!question]- 가우시안 R(D)=½log₂(σ²/D)에서 "6dB/비트 법칙"이 나오는 과정은?
> 비트율을 1비트 늘리면 로그 인자가 2 증가, 즉 D가 4배 감소한다. MSE(전력)가 4배 줄면 SNR은 `10log₁₀4 ≈ 6dB` 개선. 곧 양자화 비트 하나가 SNR 약 6dB에 해당한다.

## 연습문제

> [!example]- 문제: 분산 σ²=16인 가우시안 소스를 MSE 왜곡 D=1로 압축할 때 최소 비트율 R(D)를 구하라.
> **풀이**
> `R(D) = ½ log₂(σ²/D) = ½ log₂(16/1) = ½ log₂16 = ½·4 = 2 비트/샘플`.
> 왜곡을 D=4로 늘리면 `½log₂(16/4) = ½·2 = 1`비트로, D를 4배 허용할 때마다 1비트씩 절약됨을 확인할 수 있다.

> [!example]- 문제: 이산 소스에서 R(0)=H(X)임을 직관적으로 설명하고, 손실 압축이 왜 H 아래로 갈 수 있는지 논하라.
> **풀이**
> D=0이면 왜곡 없음, 곧 X̂=X를 완벽 복원해야 한다. 이때 필요한 최소 비트는 무손실 하한인 엔트로피 H(X)이므로 `R(0)=H(X)`(소스 코딩 정리와 일치).
> D>0을 허용하면 사람이 못 느끼는 차이(예: 고주파 계수)를 버려 X̂을 근사만 하면 되고, 이는 I(X;X̂)를 줄여 R(D)<H로 내려간다. R(D)는 D에 대해 단조 감소·볼록.

## 파인만

> [!note]- 백지에 R(D) 곡선을 그리고 R(0), 단조성, C와의 쌍대성을 설명하라.
> **점검 포인트**: (1) R(D)=min I(왜곡 제약)의 정의와 R(0)=H, (2) D↑ → R↓의 단조 감소·볼록성, (3) 용량(max I, 중복 추가)과 R(D)(min I, 중복 제거)의 쌍대 구조.
