# 결합·조건부 엔트로피 (Joint and Conditional Entropy)

## 한 줄 요약

두 변수를 함께 보는 결합 엔트로피 H(X,Y)와, 한쪽을 알 때 남는 불확실성인 조건부 엔트로피 H(Y|X). 둘을 잇는 것이 **chain rule** `H(X,Y)=H(X)+H(Y|X)`. 조건은 결코 불확실성을 늘리지 않는다(`H(Y|X)≤H(Y)`) - 정보는 손해가 없다.

## 왜 필요한가

- 두 변수의 정보량을 어떻게 합치나
- "X를 알면 Y가 얼마나 덜 불확실해지나"
- 상호정보량([[mutual-information]])의 토대

## 결합 엔트로피

두 변수를 **한 덩어리로** 본 불확실성:

```
H(X,Y) = -Σ p(x,y) log₂ p(x,y)
```

- (X,Y) 쌍 하나를 알려면 평균 몇 비트 ([[entropy-and-information]]의 정의를 쌍에 적용)
- X,Y 독립이면 `H(X,Y)=H(X)+H(Y)` (최대), 종속이면 그보다 작음

## 조건부 엔트로피

**X를 안 뒤에 남는 Y의 불확실성**:

```
H(Y|X) = Σ p(x) H(Y|X=x)      (각 x에서의 Y 엔트로피를 x로 평균)
       = -Σ p(x,y) log₂ p(y|x)
```

- "X를 이미 안다. 그래도 Y를 알려면 추가로 몇 비트?"
- X가 Y를 완전히 결정하면 H(Y|X)=0, X가 Y와 무관하면 H(Y|X)=H(Y)

## Chain Rule (핵심)

결합을 조건부로 분해:

```
H(X,Y) = H(X) + H(Y|X)
       = H(Y) + H(X|Y)
```

- 직관: "(X,Y) 다 알기 = X 먼저 알고 + X 안 상태에서 Y 알기"
- n개로 일반화: `H(X₁..Xₙ) = Σ H(Xᵢ | X₁..Xᵢ₋₁)`
- 확률의 곱셈 규칙(math/[[probability-basics]])이 log로 덧셈이 된 것

## 코드로 확인

날씨 X(맑음/비) × 우산 Y(챙김/안챙김) 결합분포로 검증:

```python
joint = {("sun","no"):0.4, ("sun","yes"):0.1,
         ("rain","no"):0.1, ("rain","yes"):0.4}
# H(Y|X) = Σ p(x) H(Y|X=x)
```

실행:
```
H(X)      = 1.0000
H(Y)      = 1.0000
H(X,Y)    = 1.7219
H(Y|X)    = 0.7219

chain rule 검증: H(X)+H(Y|X) = 1.7219  == H(X,Y) = 1.7219
H(Y|X) <= H(Y)?  0.7219 <= 1.0000  (조건이 불확실성 줄임)
상호정보 I(X;Y) = H(Y)-H(Y|X) = 0.2781
```

- **chain rule 성립**: `1.0 + 0.7219 = 1.7219 = H(X,Y)` ✓
- **날씨를 알면 우산이 덜 불확실**: H(Y)=1.0 → H(Y|X)=0.72 (비 오면 챙길 확률↑이라 예측 쉬워짐)
- 줄어든 만큼(0.278)이 곧 상호정보량 → [[mutual-information]]

## 조건은 해롭지 않다 (핵심 부등식)

```
H(Y|X) ≤ H(Y)      (등호는 X,Y 독립일 때만)
```

- **정보를 얻으면 불확실성은 늘지 않는다** (평균적으로)
- 주의: **특정** x에선 H(Y|X=x) > H(Y) 가능 (그 관측이 헷갈리게 함) - 하지만 **평균 내면** 절대 안 늘어남
- 증명은 상호정보 ≥ 0에서 나옴 ([[mutual-information]]의 KL 비음수성)

## 왜 중요한가

- **상호정보량의 정의 재료**: `I(X;Y)=H(Y)-H(Y|X)` → [[mutual-information]]
- **채널 분석**: 수신(Y)에서 송신(X)의 불확실성 H(X|Y)이 오류의 척도 → [[channel-capacity]]
- **ML**: 특성 X가 라벨 Y의 불확실성을 얼마나 줄이나 = 정보 이득(결정 트리 분할 기준, ai-ml/[[ml-fundamentals]])

## 셀프 체크

> [!question]- chain rule `H(X,Y)=H(X)+H(Y|X)`를 직관으로 설명하면?
> (X,Y) 쌍을 다 아는 일은 "먼저 X를 알고, 그다음 X를 안 상태에서 Y를 아는" 두 단계로 쪼갤 수 있다. 확률의 곱셈규칙 `p(x,y)=p(x)p(y|x)`가 로그를 거치며 덧셈이 된 것.

> [!question]- `H(Y|X) ≤ H(Y)`가 항상 성립하는데, 특정 x에서 `H(Y|X=x) > H(Y)`가 가능한 이유는?
> 부등식은 x에 대해 평균 낸 조건부 엔트로피에 관한 것이다. 어떤 특정 관측 x는 오히려 Y를 더 헷갈리게 만들 수 있지만(그 항이 H(Y)보다 큼), p(x)로 가중 평균하면 절대 늘지 않는다. 증명은 상호정보 I(X;Y)≥0에서 나온다.

> [!question]- X가 Y를 완전히 결정할 때와 X,Y가 독립일 때 H(Y|X)는 각각 얼마인가?
> X가 Y를 완전 결정하면 H(Y|X)=0(X를 알면 Y에 불확실성 없음). 독립이면 H(Y|X)=H(Y)(X를 알아도 Y 불확실성 그대로). 실제 종속 관계는 그 사이 값.

## 연습문제

> [!example]- 문제: 결합분포가 `p(x₁,y₁)=½, p(x₁,y₂)=¼, p(x₂,y₁)=¼, p(x₂,y₂)=0`일 때 H(X), H(X,Y), H(Y|X)를 구하고 chain rule을 검증하라.
> **풀이**
> 주변분포 `p(x₁)=¾, p(x₂)=¼` → `H(X) = -(¾log₂¾ + ¼log₂¼) = 0.811`.
> 결합 `H(X,Y) = -(½log₂½ + ¼log₂¼ + ¼log₂¼) = 0.5+0.5+0.5 = 1.5`.
> chain rule로 `H(Y|X) = H(X,Y) - H(X) = 1.5 - 0.811 = 0.689 비트`.
> 검산: `H(Y|X=x₁)`은 조건부 `(⅔,⅓)` → 0.918, 가중 ¾; `H(Y|X=x₂)=0`, 가중 ¼ → `0.75·0.918 = 0.689` ✓.

> [!example]- 문제: `H(X,Y) ≤ H(X) + H(Y)`이고 등호가 X,Y 독립일 때만임을 보여라.
> **풀이**
> chain rule로 `H(X,Y) = H(X) + H(Y|X)`. "조건은 해롭지 않다"에서 `H(Y|X) ≤ H(Y)`. 두 식을 합치면 `H(X,Y) ≤ H(X) + H(Y)`. 등호는 `H(Y|X)=H(Y)`, 곧 X,Y가 독립일 때만(상호정보 I(X;Y)=0). ∎

## 파인만

> [!note]- 백지에 chain rule과 "조건은 해롭지 않다"를 그림 없이 말로 유도해보라.
> **점검 포인트**: (1) H(X,Y)=H(X)+H(Y|X)가 확률 곱셈규칙의 로그판이라는 것, (2) H(Y|X)가 "각 x의 Y엔트로피를 p(x)로 평균"이라는 정의, (3) H(Y|X)≤H(Y)는 평균 부등식이지 x별 부등식이 아니라는 것.
