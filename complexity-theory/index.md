# 계산 복잡도 syllabus

기준: **Arora-Barak** (Computational Complexity) + Sipser 3부. 오토마타 (automata/)의 복잡도 맛보기([[complexity-classes]], [[space-complexity]])를 본격적으로 파고드는 과목.

## 1. 시간 복잡도

- [x] [[p-and-np]] - P, NP, 검증자 정의, NP가 왜 중요한가
- [x] [[np-completeness]] - Cook-Levin 정리, SAT, 대표 NP-완전 문제
- [x] [[reductions-and-hardness]] - 다항 시간 환원, NP-hard vs NP-complete

## 2. 공간 복잡도

- [x] [[space-classes]] - L, NL, PSPACE, Savitch 정리, TQBF
- [x] [[hierarchy-theorems]] - 시간·공간 계층 정리, 대각화

## 3. 다루기 힘든 문제 대처

- [x] [[approximation-algorithms]] - 근사 비율, PTAS, APX (algorithms/와 연결)
- [x] [[randomized-complexity]] - BPP, RP, 무작위성의 힘
- [x] [[beyond-np]] - 다항 계층 PH, #P, 상호작용 증명 맛보기
