# 신경망 (Neural Networks)

## 한 줄 요약

선형 모델을 층층이 쌓고 사이에 **비선형 활성화**를 끼워 곡선·복잡한 패턴을 학습하는 것. 학습은 **역전파**(연쇄법칙으로 각 가중치의 기울기를 뒤에서 앞으로 계산) + 경사하강법. 비선형이 없으면 아무리 쌓아도 그냥 선형 모델이다.

## 왜 필요한가

- 선형 모델의 한계를 어떻게 넘나 ([[linear-models]])
- 역전파가 실제로 뭔가 (연쇄법칙)
- 왜 비선형 활성화가 필수인가

## 퍼셉트론 → MLP

```
퍼셉트론(뉴런 1개): ŷ = 활성화(wx + b)   = 선형모델 + 활성화
MLP(다층): 뉴런을 층으로 쌓음
  입력층 → 은닉층(들) → 출력층
```

- 각 뉴런은 **가중합 + 활성화** ([[linear-models]]의 선형 유닛 + 비선형)
- 층을 쌓아 점점 추상적 특성 학습 (저수준→고수준)

## 비선형 활성화가 핵심

**활성화 없으면 여러 층 = 한 층** (선형의 합성은 선형):

```
선형만: W₂(W₁x) = (W₂W₁)x = W'x   ← 그냥 선형 하나
비선형 σ 끼움: W₂·σ(W₁x)          ← 곡선 학습 가능
```

활성화 함수:
- **ReLU** `max(0,x)`: 현대 표준. 간단·기울기 소실 완화 ([[training-dynamics]])
- **시그모이드** `1/(1+e⁻ˣ)`: 0~1, 출력층 확률용 (은닉층엔 기울기 소실로 잘 안 씀)
- **tanh**: -1~1
- **비선형이 표현력의 원천** - 이게 신경망을 강력하게 (보편 근사 정리)

## 역전파 (backpropagation)

각 가중치가 손실에 주는 영향(기울기)을 **연쇄법칙으로 뒤에서 앞으로** 계산:

```
forward:  입력 → 층층이 계산 → 출력 → 손실
backward: 손실의 기울기를 출력→입력 방향으로 전파
          각 층에서 ∂L/∂W 계산 (연쇄법칙)
```

연쇄법칙(math/[[derivatives-and-gradients]])이 핵심:
```
∂L/∂W₁ = ∂L/∂출력 · ∂출력/∂은닉 · ∂은닉/∂W₁   (곱으로 연결)
```

- **왜 뒤에서 앞으로**: 앞 층 기울기가 뒤 층 기울기에 의존 → 재사용 (동적계획법처럼 효율)
- 계산하면 경사하강법으로 갱신 ([[linear-models]]) → 이게 학습

## 코드로 확인: XOR

XOR은 **선형 분리 불가**([[linear-models]]로 못 풂) → 은닉층 필요. 역전파 직접 구현:

```python
# 2입력 -> 2은닉 -> 1출력, 역전파로 XOR 학습
do  = (o-y)*o*(1-o)                    # 출력층 기울기
dh  = [do*W2[j]*h[j]*(1-h[j]) for j]   # 연쇄법칙으로 은닉층 전파
# dW2, dW1 계산 후 경사하강 갱신
```

실행 결과 (10000 epoch 후):
```
XOR 학습 결과:
  [0, 0] -> 0.019 (정답 0)
  [0, 1] -> 0.984 (정답 1)
  [1, 0] -> 0.984 (정답 1)
  [1, 1] -> 0.017 (정답 0)
```

- 선형 모델이 **절대 못 하는** XOR을, 은닉층 하나로 학습 (0/1을 거의 정확히)
- 은닉층이 XOR을 선형 분리 가능한 공간으로 변환한 것
- **주의**: 초기값 운에 따라 local minima에 빠지기도 (seed 바꾸니 수렴) → [[training-dynamics]]의 초기화 문제

## 왜 신경망인가

- **비선형·복잡 패턴**: 선형이 못 하는 것 (이미지, 언어)
- **자동 특성 학습**: 손으로 특성 안 짜도 층이 학습 (표현 학습)
- **확장성**: 크게·깊게 → 더 복잡한 함수 (딥러닝)
- 이 기본이 CNN/RNN([[cnn-rnn-overview]])·트랜스포머([[attention-and-transformers]])의 토대

## 한계·주의

- **데이터·계산 많이 필요**: 작은 데이터엔 과적합 ([[ml-fundamentals]])
- **블랙박스**: 해석 어려움 (선형 모델과 대조)
- **학습 불안정**: 기울기 소실/폭발, local minima → [[training-dynamics]]

## 셀프 체크

> [!question]- 비선형 활성화가 없으면 왜 층을 쌓아도 소용없나?
> 선형의 합성은 선형이라, W₂(W₁x)=(W₂W₁)x=W'x로 결국 한 층과 같다. 사이에 비선형 σ를 끼워 W₂·σ(W₁x)가 되면 비로소 곡선·복잡한 패턴을 학습할 수 있다(보편 근사).

> [!question]- 역전파는 무엇을 어떤 방향으로 계산하나?
> 각 가중치가 손실에 주는 영향(∂L/∂W)을 연쇄법칙으로 출력→입력 방향(뒤에서 앞으로) 계산한다. 앞 층 기울기가 뒤 층 기울기에 의존하므로 뒤에서부터 계산해 재사용하면 효율적이다(동적계획법처럼).

> [!question]- ReLU가 은닉층 표준이고 시그모이드가 은닉층에 잘 안 쓰이는 이유는?
> ReLU max(0,x)는 간단하고 양의 구간에서 기울기를 1로 유지해 기울기 소실을 완화한다. 시그모이드는 양 끝에서 기울기가 0에 가까워 깊은 망에서 소실을 일으킨다. 시그모이드는 확률이 필요한 출력층에 주로 쓴다.

> [!question]- XOR이 신경망에서 왜 상징적 예제인가?
> XOR은 선형 분리가 불가능해 선형 모델로는 못 푼다. 은닉층 하나가 입력을 선형 분리 가능한 공간으로 변환해 풀어내므로, 비선형 은닉층의 필요성을 보여준다.

## 연습문제

> [!example]- 문제: 출력 o(시그모이드), 정답 y, 손실을 L=½(o−y)²로 두면 출력층 기울기 do가 (o−y)·o(1−o)임을 유도하라.
> **풀이**
> o=σ(z)라 하면 dL/do = (o−y). 시그모이드 미분 dσ/dz = o(1−o).
> 연쇄법칙: dL/dz = dL/do · do/dz = (o−y)·o(1−o).
> 이것이 노트 코드의 `do = (o-y)*o*(1-o)`다. 은닉층으로는 다시 dh = do·W2·h(1−h)로 연쇄법칙을 이어 전파한다.

> [!example]- 문제: 2층 망 x→a=σ(z₁)→o=σ(z₂), 손실 L에 대해 ∂L/∂W₁을 연쇄법칙 곱으로 써라. 왜 "뒤에서 앞으로" 계산하는 게 효율적인지 설명하라.
> **풀이**
> ∂L/∂W₁ = ∂L/∂o · ∂o/∂a · ∂a/∂z₁ · ∂z₁/∂W₁. 즉 출력 쪽 항들을 순서대로 곱해 입력 층까지 내려간다.
> ∂L/∂W₂ 계산에서 이미 ∂L/∂o를 구했고, ∂L/∂W₁은 그 값을 재사용해 앞으로 더 곱하기만 하면 된다. 뒤 층 기울기를 앞 층이 공유하므로 뒤에서부터 한 번에 계산하면 중복 계산이 없어 효율적이다.
