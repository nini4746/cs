# 선형 모델 (Linear Models)

## 한 줄 요약

입력의 가중합으로 예측하는 가장 단순한 학습 모델 - 선형회귀(연속값)와 로지스틱회귀(확률/분류). 학습은 **손실 함수를 경사하강법으로 최소화**하는 것이고, 이 최적화 틀이 신경망까지 그대로 확장된다. ML의 "hello world".

## 왜 필요한가

- 모델이 실제로 어떻게 "학습"하나 (경사하강법)
- 손실 함수가 뭔가
- 회귀와 분류의 차이

## 선형회귀

**입력의 가중합**으로 연속값 예측:

```
ŷ = w₁x₁ + w₂x₂ + ... + b   (w=가중치, b=편향)
```

- 목표: 데이터에 제일 잘 맞는 `w, b` 찾기
- 손실: **MSE**(평균제곱오차) `(1/n)Σ(ŷ-y)²` - 예측과 정답 차이의 제곱 평균

## 로지스틱회귀 (분류)

선형 출력을 **확률로** 짜냄 → 분류:

```
z = wx + b
ŷ = σ(z) = 1/(1+e⁻ᶻ)   (시그모이드: 실수 → 0~1 확률)
```

- **분류인데 이름은 회귀** (확률을 회귀)
- 손실: **크로스 엔트로피** (MSE 아님) - 확률 예측엔 이게 맞음
- 결정 경계는 선형 (`z=0` 지점)

## 경사하강법 (핵심)

**손실을 줄이는 방향으로 파라미터를 조금씩 이동** - 모든 학습의 엔진:

```
1. 예측 → 손실 계산
2. 손실의 기울기(gradient) 계산 (각 파라미터가 손실에 주는 영향)
3. 기울기 반대 방향으로 이동:  w ← w - lr·∂L/∂w
4. 반복
```

- **기울기 = 가장 가파른 오르막** → 반대로 가면 손실↓ (내리막)
- **학습률(lr)**: 걸음 크기. 너무 크면 발산, 너무 작으면 느림 ([[training-dynamics]])
- math/[[derivatives-and-gradients]]의 기울기가 여기 실전

## 코드로 확인

`y = 2x + 1`을 경사하강법으로 학습 (파라미터를 0에서 시작해 찾아냄):

```python
xs = [0,1,2,3,4]; ys = [1,3,5,7,9]   # 정답 w=2, b=1
w, b, lr = 0.0, 0.0, 0.01
n = len(xs)
for step in range(2000):
    dw = db = 0.0
    for x,y in zip(xs,ys):
        err = (w*x+b) - y            # 예측 - 정답
        dw += 2*err*x/n              # ∂MSE/∂w
        db += 2*err/n               # ∂MSE/∂b
    w -= lr*dw                       # 기울기 반대로 이동
    b -= lr*db
```

실행 결과 (0에서 시작 → 정답으로 수렴):
```
step    0: w=0.2800 b=0.1000 mse=24.752400
step  100: w=2.0509 b=0.8548 mse=0.007063
step  500: w=2.0046 b=0.9868 mse=0.000059
step 1999: w=2.0000 b=1.0000 mse=0.000000
```

- 손실이 24.75 → 0으로, `w,b`가 정답 2,1로 스스로 수렴
- **이게 "학습"의 실체**: 손실의 기울기를 따라 파라미터 조정

## 왜 선형 모델이 중요한가

- **학습의 원리를 다 담음**: 손실→기울기→갱신 (신경망도 같은 틀, [[neural-networks]])
- **해석 가능**: 각 가중치가 특성의 영향 (블랙박스 아님)
- **강한 베이스라인**: 단순해도 잘 되는 경우 많음 (오컴의 면도날)
- **빠름·확장**: 큰 데이터에도 (선형)

## 한계

- **선형 관계만**: 곡선·복잡한 패턴 못 잡음 → 특성공학 or 신경망
- 특성 간 상호작용 수동 추가 필요
- 이 한계가 [[neural-networks]](비선형 활성화로 곡선 학습)의 동기
