# NISQ와 하드웨어 (NISQ and Hardware)

## 한 줄 요약

NISQ(Noisy Intermediate-Scale Quantum)는 큐빗 수가 수십~수천이지만 오류 정정이 안 되는 **현재 세대** 양자 컴퓨터를 가리킨다(Preskill, 2018). 물리 구현은 초전도(superconducting)와 이온 트랩(ion trap)이 주류이며, 각각 큐빗 수·게이트 속도·충실도(fidelity)·연결성에서 장단점이 다르다. 결어긋남과 게이트 오류 때문에 정정([[quantum-error-correction]]) 없이는 짧은 회로만 돌릴 수 있어, "양자 우위"(quantum advantage) 주장과 실용성 사이의 간극이 이 시대의 핵심 논쟁이다.

## 왜 필요한가

- 이론(알고리즘)과 현실(물리 소자) 사이의 격차를 이해
- 왜 쇼어를 아직 못 돌리는지, 언제쯤 가능한지의 근거 → [[shor-algorithm]]
- 큐빗 수·충실도·연결성이 알고리즘 선택을 제약
- 투자·과대광고 속에서 "무엇이 진짜 가능한가"를 판별하는 감각

## NISQ 시대 정의

| 특성 | 내용 |
|---|---|
| 규모 | 수십~수천 큐빗 (intermediate-scale) |
| 노이즈 | 오류 정정 없음, 게이트마다 오류 누적 (noisy) |
| 회로 깊이 | 결어긋남 전까지 수십~수백 게이트 |
| 알고리즘 | 얕은 변분 회로(VQE, QAOA) 위주 |

- Preskill(2018)이 명명 - "곧 오지만 결함 허용은 아직"인 과도기
- 논리 큐빗 없이 물리 큐빗을 직접 사용

## 물리 구현 비교

| 방식 | 큐빗 인코딩 | 장점 | 단점 |
|---|---|---|---|
| 초전도(superconducting) | 조셉슨 접합 회로 | 빠른 게이트, 반도체 공정 확장 | 짧은 수명, 극저온(~10mK), 국소 연결 |
| 이온 트랩(ion trap) | 이온 에너지 준위 | 긴 수명, 높은 충실도, 전연결 | 느린 게이트, 확장 어려움 |
| 광자(photonic) | 광자 편광/경로 | 상온, 통신 친화 | 2큐빗 게이트 확률적 |
| 중성 원자(neutral atom) | 리드베르그 원자 | 확장성, 재배치 유연 | 신생, 게이트 충실도 개선 중 |

- 초전도(IBM, Google) vs 이온 트랩(IonQ, Quantinuum)이 대표 경쟁 구도
- 중성 원자(QuEra 등)가 최근 급부상

## 핵심 성능 지표

| 지표 | 의미 |
|---|---|
| 큐빗 수 | 규모 (많다고 좋은 게 아님) |
| 게이트 충실도(fidelity) | 게이트가 얼마나 정확한가 (2큐빗이 병목, ~99.x%) |
| T1 / T2 | 큐빗 수명 (에너지/위상 결어긋남 시간) |
| 연결성(connectivity) | 어떤 큐빗쌍이 직접 게이트 가능한가 |
| 양자 볼륨(quantum volume) | 큐빗 수·충실도·연결성 종합 지표 |

- **충실도가 큐빗 수보다 중요** - 오류가 곱해지며 깊은 회로를 막음
- 예: 2큐빗 게이트 오류 1%면 100게이트 후 신뢰도 붕괴
- 연결성 낮으면 SWAP 게이트로 우회 → 회로 깊이 증가

## 결어긋남과 오류 예산

```
유효 회로 깊이 ≈ (큐빗 수명) / (게이트 시간)
```

- 결어긋남(decoherence): 환경과의 얽힘으로 양자성 소실 → [[quantum-error-correction]]
- 모든 게이트·측정·유휴가 오류를 더함 → "오류 예산" 안에서만 계산
- 정정 없는 NISQ에선 이 예산이 수백 게이트 수준

## 양자 우위 논쟁

특정 인위적 과제에서 고전 슈퍼컴퓨터 추월 주장:

| 사례 | 주장 | 반론 |
|---|---|---|
| Google Sycamore (2019) | 53큐빗, 랜덤 회로 샘플링 200초 vs 1만년 | 개선된 고전 알고리즘이 수일로 단축 |
| 중국 Jiuzhang (광자) | 보손 샘플링 | 실용 무의미 과제 |

- **양자 우위(quantum supremacy/advantage)**: 고전이 사실상 못 하는 걸 양자가 함
- 문제: 시연 과제가 **실용성 없음**(rigged benchmark 성격) + 고전 알고리즘이 계속 따라잡음
- 진짜 목표는 "유용한(useful) 양자 우위" - 아직 도달 못 함

## NISQ 알고리즘

정정이 없으니 얕은 회로로 노이즈를 견디는 방식:

- **VQE**(변분 양자 고유해): 화학 분자 바닥상태 에너지, 양자-고전 하이브리드
- **QAOA**: 조합 최적화 근사
- 특징: 얕은 파라미터 회로 + 고전 최적화 루프 → 노이즈 부분 완화
- 한계: 이들조차 명확한 실용 우위는 미증명

## 전망

- 결함 허용까지는 수백만 물리 큐빗·임계값 이하 충실도 필요 → [[quantum-error-correction]]
- 로드맵상 초전도·이온 트랩 모두 큐빗 확장 + 충실도 개선 병행
- 쇼어급 응용은 수년~십수 년 이후 전망, 그전까지 PQC 전환 진행 → cryptography/[[public-key-crypto]]
